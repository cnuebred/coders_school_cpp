# linux commands

```
    echo - wyświetla coś po komendzie
    `date` - wyświetla date
```

```
    more - wyświetla zawartość pliku wyjście 'q'
```

```
    less - wyświetla zawartość pliku
    po '/' szukamy w pliku
```

```
    clear - czyści ekran
```

```
    ls - wyświetla wszystko w katalogu
    flaga  -l - porządkuje
```

```
    man <komenda> - instrukcja użycia 
```

```
    n - przeskakiwanie między stronami
```

```
    head i tail pokazuje 10 pierwszych i ostatnich linijek dokumentu
    Flaga: 
    -n <liczba linijek>
```

```
    touch - tworzy plik
```

```
    mkdir <nazwa> - tworzenie katalogu
```
```
    rm -r <nazwa> - usuwanie katalogu/pliku   
    -f - force (wymuś)
```
```
    which - pokazuje gdzie leży program ls
```
```
    chmod - uprawnienia do pliku
```
```
    ocular - otwiera image
```
```
    vim - edytor plików 
    wyjście - to esc :wq - zapisanie zmian i wyjście 
    esc :q - zabezpieczenie i trzeba dopisać !
    vimtutor - tutorial vima
    flaga: 
    <plik> +<linia kodu> - przeskakuje do danej lini kodu
```

```
    cp <file> <plik do którego kopiujemy>- kopjuje plik
```
```
    mv copy.txt ../backup.txt
```
```
    find <katalog> [parametry] - szukanie pliku
    parametry:
    -name file.txt
    -iname zawiera coś z flagi
    katalog:/ wszystko
            . katalog
            ~ katalog domowy
```

```
    przekierowania > np:

    echo "cos" > file
    cat file
```
```
    wc - pojazuje liczbe znaków/słów w plikach
```

```
    szukanie w logach 
    history - pokazyje wszystkie komendy 
    ctrl + r
```

```
    przekierowanie błędów do dev/null 
    2> tylko błędy 
    &> wszytko co wywołujemy
```

```
    od -SPRAWDZIC
    od -td1 -N10 /dev/random
```

```
    vimdiff <plik> <plik> - porównuje dwa pliki
```

```
    hexdump - wyświetla wszystko w hex
```

```
    jobs - wyświetla pliki w tle działające
    fg - przywraca proces 
    ctrl + z - daje procesy w tło
```

```
    top pokazuje procesy
```

```
    grep - wyszukiwanie lini w plikach
```
